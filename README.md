## Run Locally

### 1. Clone the project

```bash
  git clone https://github.com/techCarpenter/topple.git topple
```

### 2. Go to the project directory

```bash
  cd topple
```

### 3. Install dependencies

```bash
  npm install
```

### 4. Start the server

```bash
  npm run serve
```
